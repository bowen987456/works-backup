# OpenClaw Heartbeat Checklist

## [每30分钟]

- 检查模型/API 连接是否正常

## [每日2–3次]
- 系统监控：内存、CPU、磁盘（>85%告警）
- 工作区：清理超大文件（>2000 tokens）
- OpenClaw 版本与安全更新检查
- 错误日志快速巡检

## [每日1次（晚间）]
- 整理记忆：提炼日志到 MEMORY.md，清理过期
- Git 同步：自动提交变更（无变更跳过）
- 生成极简日报：完成/阻塞/明日重点

## [每3–5次心跳]
- 技能：自动安装未装的关键技能
- 配置校验：config.json 关键项正常
- 清理：30天前日志、无用缓存
